5. Accessibility Checklist

Contrast

ContextLight ModeDark ModeRatio
Body textpaper-80 on paper-05paper-05 on paper-9514.17:1 / 17.44:1
Secondary textpaper-60 on paper-05paper-20 on paper-95~7:1 / ~4.8:1
Code textpaper-80 on paper-00paper-05 on paper-8015:1 / 12:1

PROHIBITED: paper-40 as text on paper-95 or paper-90. Contrast ratio 1.45:1 — 68% failure against 4.5:1 requirement.

Focus

/* Always visible focus indicators */
:focus-visible {
  outline: 2px solid var(--amber);
  outline-offset: 2px;
}
/* NEVER outline: none without alternative */

ARIA

  • Commit button: role="button", aria-disabled, aria-label="Commit entry"
  • Search: role="search", aria-expanded, aria-live="polite"
  • Intent dot: aria-hidden="true" (announced via parent label)
  • Theme toggle: aria-label="Toggle dark mode"

Keyboard

Tab / Shift+Tab, Enter/Space for buttons, Cmd+Enter to commit, Escape to close.

Reduced Motion

@media (prefers-reduced-motion: reduce) {
  * { animation-duration: 0.01ms !important; transition-duration: 0.01ms !important; }
}

7. Implementation Checklist

  1. Add design-tokens-swift.swift to Xcode project
  2. Register Space Mono and IBM Plex Sans in Info.plist
  3. Implement LogColors semantic functions (textPrimary, backgroundPrimary, etc.)
  4. Wire @Environment(\.colorScheme) in all views
  5. Add @Environment(\.accessibilityReduceMotion) wherever animation exists
  6. Verify 44pt minimum touch targets on all interactive elements
  7. Verify focus indicators (amber outline) on all focusable elements
  8. Test dark mode: no paper-40 on dark backgrounds
  9. Test signal colors: same hex in both modes
  10. Test reduced motion: all animations become instant
  11. Test Dynamic Type: xSmall through xxxLarge
  12. Verify idle trace: appears instantly, no animation

8. Anti-Patterns

Don'tDo
Hardcode hex valuesUse LogColors / CSS variables
Use paper-40 for dark mode textUse paper-05 (body) or paper-20 (secondary)
Invert signal colors in dark modeKeep signals constant
Use spring/elastic animationsUse mechanical easing only
Animate the idle traceRender at final height instantly
Use emoji in UIUse SVG with stroke="currentColor"
Use IBM Plex MonoUse Space Mono for mechanism
Add decorative animationOnly 6 animations allowed
Touch targets < 44pxmin-width/height: 44px on all interactive elements
Suppress focus outlinesAlways show 2px amber outline
